# Installation

Create python virtual environment:
```
$ python -m venv venv
```

Activate virtual environment (in powershell):
```
$ .\venv\Scripts\Activate.ps1
```

Install requirements file:
```
pip install -r requirements-BASE.txt
pip install -r requirements-CRAWLING.txt

```
